NAME

cryptexctl dump-trust-cache — print the content of a trust cache.

SYNOPSIS

cryptexctl dump-trust-cache dump-trust-cache [-i | --im4p] TRUST-CACHE-PATH

DESCRIPTION

Print the content of a trust cache. The output will be of the form:

<cdhash> [<occurance count>] <flags>

The same code directory hash (cdhash) may occur multiple times in a single trust cache, and only the number of occurrences are printed.

OPTIONS

The following options are accepted by cryptexctl dump-trust-cache dump-trust-cache

-i | --im4p
Indicate that the file is a wrapped Image4 payload
TRUST-CACHE-PATH
Path to the trust cache to dump
