CRYPTEXCTL-LIST(1) General Commands Manual CRYPTEXCTL-LIST(1)

NAME

cryptexctl list — list installed cryptexes

SYNOPSIS

cryptexctl list list [-l | --lossy]

DESCRIPTION

List the installed cryptexes, either on the host (if the host has a cryptex management daemon) or on a device specified in the top level command (see cryptexctl(1)).

OPTIONS

List of flags and their descriptions:

-l | --lossy
Fetch the list of installed cryptexes in a lossy way which will not communicate with cryptexd(8). Some information may not be available or up to date.

SEE ALSO

cryptexctl(1), cryptexctl-create(1), cryptexctl-install(1), cryptexd(8)
