FTMSTOOL(8) System Manager's Manual FTMSTOOL(8)

NAME

ftmstool — interfaces with facetimemessagestore to manage and inspect carrier voicemails, Live Voicemails, and video messages.

SYNOPSIS

takes no arguments, and users should not launch it manually.

DESCRIPTION

ftmstool is program that can be used to create, read, edit, and delete messages stored by facetimemessagestored. For available commands, use ftmstool -h.

SEE ALSO

facetimemessagestore(8),

HISTORY

ftmstool first appeared in iOS 17.0, watchOS 10.0, and macOS 16.0
