NAME

remotesoftwareupdated — Remote Software Update daemon

DESCRIPTION

remotesoftwareupdated is a daemon that facilities paired devices to perform a remote software update over the network.

There are no configuration options to remotesoftwareupdated. Users should not run this remotesoftwareupdated manually.
