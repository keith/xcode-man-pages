OS_CROSS_ARCH_IS_SUPPORTED(3) Library Functions Manual OS_CROSS_ARCH_IS_SUPPORTED(3)

NAME

os_cross_arch_is_supported — determine if cross-architecture compatibility is supported

SYNOPSIS

#include <os/arch/arm64.h>

bool
os_cross_arch_is_supported(os_cross_arch_t arch);

DESCRIPTION

The os_cross_arch_is_supported() function returns whether the kernel provides the set of cross-architecture compatibility features required by a user-space compatibility layer to run processes built for the CPU architecture identified by arch via translation.

A true return value does not imply that any particular compatibility layer is installed, enabled, or functional on the running system; it only reports whether the underlying kernel features that such a layer depends on are present.

The arch argument must be one of the following constants:

OS_CROSS_ARCH_X86_64
The x86_64 (Intel/AMD 64-bit) architecture.

Any other value of arch, including values that may be defined by future SDKs but are not recognized by the running system, causes the function to return false.

The return value is constant for the lifetime of the system and may be safely cached by callers.

RETURN VALUES

The os_cross_arch_is_supported() function returns true if cross-architecture compatibility for arch is supported, or false otherwise.
