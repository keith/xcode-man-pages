NAME

proactiveeventtrackerd — a process triggered by a daily XPC activity to transform and upload metrics stored in the aggregate state file

DESCRIPTION

proactiveeventtrackerd The ProactiveEventTracker is a framework used by Proactive Siri to efficiently log aggregated counters and metrics into a memory mapped file. proactiveeventtrackerd is a process triggered by a low-priority daily XPC activity to transform the data structures stored in the file into protobuf messages, and uploading these messages for diagnostics and analytics
