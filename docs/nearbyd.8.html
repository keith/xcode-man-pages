NAME

nearbyd — The proximity daemon

SYNOPSIS

nearbyd takes no arguments, and users should not launch it manually.

DESCRIPTION

nearbyd powers spatial interaction experiences between devices using ultra wideband and other wireless technologies

FILES

/System/Library/LaunchDaemons/com.apple.nearbyd.plist
The nearbyd service's property list file for launchd(8)

SEE ALSO

launchd(8)

HISTORY

first appeared in macOS 10.14 and iOS 12.0
