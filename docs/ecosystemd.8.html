ECOSYSTEMD(8) System Manager's Manual ECOSYSTEMD(8)

NAME

ecosystemd — Collects information surrounding targeted events and triggers responses within the system.

SYNOPSIS

takes no arguments, and users should not launch it manually.

DESCRIPTION

Ecosystemd tracks when applications use certain system functionality. It can inform the user via ecosystemagent of that use. This information is used by Apple to improve its products. HISTORY ecosystemd first appeared in macOS 16
