ACCESSORYACCESSD(8) System Manager's Manual ACCESSORYACCESSD(8)

NAME

accessoryaccessd — Accessory exclusive access management

DESCRIPTION

The accessoryaccessd daemon is responsible for managing exclusive access to accessories on behalf of users.

accessoryaccessd is launched on-demand by launchd(8) and must not be invoked directly. Interaction with accessoryaccessd is done via the AccessoryAccess framework.
