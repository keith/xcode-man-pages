NAME

tapi-archive - Merge or thin text-based stub files

SYNOPSIS

tapi archive [--help] [--version]

tapi archive --info <file>

tapi archive --verify-arch <architecture> <file>

tapi archive --extract <architecture> <file> -o <file>

tapi archive --remove <architecture> <file> -o <file>

tapi archive --merge <files> -o <file>

DESCRIPTION

The archive command is used to inspect and manipulate text-based stub files.

OPTIONS

--help

Prints the list of options.

For information on the complete set of command line options, please use 'tapi archive --help'
